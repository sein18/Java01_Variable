# Variable (변수)

* __데이터 저장 단위__
  * __`비트(bit)` __: 컴퓨터가 나타내는 데이터의 저장 최소 단위로 2진수 값 하나를 저장할 수 있는 메모리공간을 뜻한다.
  * __`바이트(byte) `__: 데이터처리 또는 문자의 최소 단위로서 8개의 비트가 모여 하나의 바이트가 구성된다.

---

* __변수란__  : 메모리 공간(RAM)에 한 개의 값을 기록하기 위한 장소 공간이다.
* __변수의 자료형(Type)__ 
  * __기본형(Prinitive Type)__
    * 실제 데이터 값을 저장
    * 논리형,문자형,정수형,실수형으로 `boolean(1byte), char(2byte), byte(1byte), short(2byte), int(4byte), long(8byte), float(4byte), double(8byte)` 나눠있다.
  * __참조형(Reference Type)__
    * 데이터가 저장되어 있는 주소를 저장(객체의 주소)
    * 기본형을 제외한 String, 사용자 정의 자료형(객체)
    * 4byte의 공간을 저장공간으로 할당

---

* __변수 선언, 명명 규칙__
  * 변수 선언
    * `자료형 변수명;`  ex:)int age;
  * 변수의 명명 규칙
  * 대소문자가 구분된다. ex:) `int age; int AGE;`다른 변수이다.
  * 변수명을 예약어를 사용하면 안 된다.
  * 숫자로 시작하면 안 된다.
  * 특수문자는 `_` , `$` 만 가능하다.
  * 여러 단어로 연결되어 있을때 첫 글자는 대문자로한다.
